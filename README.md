📌 Project Overview

A modern and fully responsive Digital Tools Buying Website where users can browse premium tools, add them to cart, and manage purchases seamlessly. This project focuses on clean UI, smooth user interaction, and real-world e-commerce functionality using React.

🚀 Live Demo

🔗 Live Site: https://digi-tools-platform-plum.vercel.app/
🔗 GitHub Repo: https://github.com/bashar35790/DigiTools-Platform

🛠️ Technologies Used
⚛️ React.js
🎨 Tailwind CSS
🌼 DaisyUI
🧠 JavaScript (ES6+)
🔔 React-Toastify
📦 JSON (for product data)
✨ Key Features
🛒 1. Dynamic Cart System
Add products to cart instantly
Cart count updates in the navbar
Remove individual items from cart
“Proceed to Checkout” clears the cart
🔄 2. Product & Cart Toggle
Toggle between Products and Cart section
Default view shows product list
Cart shows empty message when no items added
🃏 3. Interactive Product Cards
Displays product details (name, price, features, tags)
“Buy Now” button updates to “Added to Cart”
Shows item count badge per product
📦 Product Data Structure

Each product includes:

id
name
description
price
period (monthly / yearly / one-time)
tag & tagType
features (array)
icon
